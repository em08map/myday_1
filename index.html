Your tasks sync across all your devices using a private file in your GitHub account. One-time setup, takes 2 minutes.
1 Go to github.com/settings/tokens/new and sign in if needed
2 Give it a name like MyDay, scroll down and tick the gist checkbox, then click Generate token
3 Copy the token (starts with ghp_) and paste it below
Your token is stored only in this browser. Your data is saved to a secret Gist in your own GitHub account — only you can see it. GitHub is free.
